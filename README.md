# K-web-extension

A web extension for the Lightspeed K-Series platform, providing an interactive demo interface.

## Overview

This project demonstrates the K-Series web extension capabilities with a responsive web interface built using Bootstrap and jQuery.

## Features

- Interactive web-based interface
- Responsive design for all screen sizes
- Bootstrap 4 styling
- jQuery functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4.5.2
- jQuery 3.2.1
